# CyberBot-POE
This repository contains my Programming 2A (PROG6221/w) POE project.  
Each section below corresponds to the assignment questions (Q1–Q8).

---

## Q1 – User Input Handling
I implemented input validation and error handling.  
At first I struggled with exceptions, but I fixed it by testing different cases.

## Q2 – String Manipulation
I used string concatenation and interpolation.  
W3Schools examples helped me understand escape characters.

## Q3 – Modular Design
I separated logic into different classes and methods.  
This made the project easier to debug and maintain.

## Q4 – Error Handling
I added try/catch blocks to handle invalid input.  
This prevents the program from crashing.

## Q5 – Visual Enhancements
I added ASCII art and color formatting to improve the console output.  
This makes the program more engaging.

## Q6 – Media Playback
I experimented with adding sound effects.  
This was tricky, but I managed to integrate basic playback.

## Q7 – Chatbot Features
I added randomization to chatbot responses.  
This makes the conversation flow more natural and varied.

## Q8 – CI Workflow Setup
I set up a GitHub Actions workflow (`dotnet-ci.yml`) so that every push checks if the project builds in .NET.  
At first it failed because I missed a dependency, but after fixing it the workflow runs.  
It still shows me an error even though it runs, so I’m working on figuring that out.  
This setup helps me make sure my code compiles before I submit.
